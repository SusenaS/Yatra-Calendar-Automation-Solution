# Yatra Calendar Automation Solution

This project automates the **Yatra.com calendar component** using **Java, Selenium WebDriver, Maven, TestNG, and the Page Object Model (POM)**.

The script selects a future date dynamically by navigating the calendar by current montha and next month, comparing the prices of both months and returing the lowest price.
